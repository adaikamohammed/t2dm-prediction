# T2DM Clinical Prediction Dashboard

**Live Demo:** `https://YOUR-USERNAME.streamlit.app` *(link after deployment)*

An XAI-driven clinical decision support tool for Type 2 Diabetes Mellitus prediction.  
Based on the MIDE 2026 research paper: *An XAI-Driven Universal Framework for Diabetes Prediction*.

## Features
- 🩺 **Real-time T2DM risk probability**
- 🧠 **SHAP Waterfall explanation** per patient
- 📊 **LightGBM + Borderline-SMOTE** (97.14% Accuracy, 70.0% Recall)

## Deploy on Streamlit Community Cloud (Free)

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "T2DM Prediction WebApp — MIDE 2026"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/t2dm-prediction.git
git push -u origin main
```

### Step 2 — Deploy
1. Go to **https://streamlit.io/cloud**
2. Sign in with GitHub
3. Click **"New app"**
4. Select your repository → Branch: `main` → Main file: `app.py`
5. Click **Deploy** → Get your public URL!

### Files Required in the Repository
```
webapp/
├── app.py           ← Main Streamlit app
├── model.pkl        ← Pre-trained LightGBM model (generated by save_model.py)
├── requirements.txt ← Python dependencies
└── README.md        ← This file
```

> ⚠️ Make sure `model.pkl` is committed to the repo (it's ~5MB).
